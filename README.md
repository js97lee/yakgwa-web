# 달빛약과 (Dalbit Yakgwa)

한국 전통 과자 약과를 소개하는 웹사이트

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## GitHub Pages 배포

1. 이 저장소를 GitHub에 푸시
2. Settings → Pages → Source: **GitHub Actions** 선택
3. `main` 또는 `master` 브랜치에 푸시 시 자동 배포

배포 URL: `https://[사용자명].github.io/yakgwa-web/`
